Пожалуйста, прочитайте Правила для авторов перед тем, как Вы выложите Ваши файлы на сервер. Выложите все файлы, которые относятся к статье.

Обязательно загрузите исходный файл (LaTeX, Word и т.п.) и PDF файл Вашей рукописи.

Важная информация: Если Вы посылаете конфиденциальную информацию, предназначенную только для редакции, обязательно оставьте комментарий к файлу или заполните форму на следующем экране.

  1. Выберите первый файл с Вашего компьютера, нажав на кнопку Browse (Обзор). Пожалуйста, добавьте комментарий к каждому файлу
  2. Нажмите кнопку 'Загрузить'. Файл появится в списке.
  3. Загрузите остальные файлы
  4. Нажмите на кнопку , чтобы изменить информацию о файле.
  5. Нажмите на кнопку , чтобы удалить файл из списка.
  6. Нажмите кнопку 'Продолжить'.
